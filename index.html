The Liverpool API

Welcome to the Liverpool API

This API contains information about every single player that made a league appearance for Liverpool football club during the 2021/22 premier league season This information includes name, nationality, appearances, clean sheets (for defensive players & goalkeepers), assist and goals scored. To consume this API, please use the following URL: https://liverpool-app.herokuapp.com/squad/{Enter player-name}. The full JSON object in this API can be found at https://liverpool-app.herokuapp.com/squad/ . This API is completely free and does not require any key

Properties and Description

Properties Description Data Type
Name Name of Player String
Shirt Number Player's jersey number Integer
Nationality Player's country of birth String
Appearances The number of matches played Integer
Assists/CleanSheets For Defenders : The number of games they played in without conceding a goal.
For Midfielders & Attackers : Passes that led directly to a goal being scored 		Integer
Goals Goals scored in the league during the season Integer
